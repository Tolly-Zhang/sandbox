# SandBox
A collection of custom-built scripts, configs, and macros intended for random, specific day-to-day tasks that I find myself doing repeatedly. These are not intended to be polished tools, but rather quick solutions to automate small tasks.

## Usage 

Each script or macro is contained in its own folder with a `README` and `config.json` file. 
- The `README` contains instructions for setup, configuration, and usage. 
- The `config.json` allows you to customize the behavior of the script or macro.
